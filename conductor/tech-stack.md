# NBA DFS Optimizer Tech Stack

## Core Language
- **Python:** High-level scripting for pipeline orchestration and data manipulation.

## Optimization & Math
- **PuLP:** Python-based modeling for Linear Programming.
- **HiGHS Solver:** High-performance, open-source solver for large-scale optimization problems.
- **NumPy & Pandas:** Efficient array processing and data manipulation for large player pools.
